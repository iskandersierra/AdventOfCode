# Day 2

[Rock Paper Scissors](https://adventofcode.com/2022/day/2)

## Programming Language

[Go](https://golang.org/)

- Install Go `go1.19 windows/amd64`

## How to run

```bash
go run ./... data/input.txt 1
go run ./... data/input.txt 2
```
