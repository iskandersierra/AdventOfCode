# Day 4

[Camp Cleanup](https://adventofcode.com/2022/day/4)

## Programming Language

[Pascal](https://www.freepascal.org/)

- Install Free Pascal `v3.2.2`

## How to run

```bash
fpc day4.pas
./day4 data/input 1
./day4 data/input 2
```
